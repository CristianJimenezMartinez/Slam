# Poetry Slam Alicante — Guía de Contenido para Eventos

## Descripción para Clasificatorios

Usar esta descripción al crear cada evento clasificatorio en el Admin:

```
Sin filtros, sin artificios. La poesía se vive en directo con la fuerza de quien se sube a decir lo que arde. El público vota. La ciudad elige.
```

## Descripción para la Final

Usar esta descripción para el evento de la FINAL:

```
La gran final. Una voz representará a Alicante en el campeonato estatal de Poetry Slam España en Cartagena. El público decide.
```
